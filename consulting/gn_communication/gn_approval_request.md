## 0. 요약

### 본 문서의 목적
바디닥터(요실금치료기 라인)의 AI 추천 노출 개선을 위해 당 연구팀이 12주에 걸쳐 진행할 페이지 개선·외부 콘텐츠·측정 인프라 항목 9건을 정리하고, 이에 필요한 GN그룹의 협조 사항을 제시합니다.

### 작업이 필요한 배경
- ChatGPT·Gemini·Perplexity 등 AI 검색 사용자가 2026년 1분기 기준 주간 8억 명을 넘어서며, 상품 검색이 AI 채팅으로 빠르게 이동 중
- 글로벌 GEO(Generative Engine Optimization) 측정 도구 11개 이상이 등장했으며, 의료기기 카테고리 전담 도구는 글로벌 0개 — 한국 시장에서 first-mover 포지셔닝 가능 (~6개월 윈도우 추정)
- 자체 측정 결과 바디닥터 PDP는 콘텐츠 측면 카테고리 중하위권으로 진단됨 (§1.2 참조)

### 기대 효과
- AI 추천에서 바디닥터 멘션률 +40~60% 개선 (Ecomtent 업계 벤치마크)
- ChatGPT Shopping rich card 트리거 가능 — 실구매 전환에 직결 (Alhena 사례)
- 12주 후 정량 Before/After 보고

### GN그룹 측 협조 범위
- **시간**: 페이지 수정 권한 위임 + 월 1회 점검 미팅(약 1시간) + CS팀 인터뷰(1시간)
- **예산**: 리뷰 캠페인 약 10만원 + 외부 콘텐츠 시딩(별도 협의) + 이미지 재촬영(별도 견적)
- **정보 제공**: 정확한 허가번호·스펙·FAQ 답변 등 페이지 작성에 필요한 자료 5건

## 1. 배경 — 작업이 필요한 이유

### 1.1 시장 변화

| 신호 | 의미 |
|---|---|
| ChatGPT 주간 검색 사용자 8억 명 이상 (2026년 1분기) | 상품 검색이 AI 채팅으로 이동 중 |
| Profound·Bluefish 등 GEO 측정 도구 11개 이상 등장 | "AI에 노출되는가"가 새 KPI로 자리잡음 |
| Schema.org Product·AggregateRating이 ChatGPT Shopping card 트리거 요건 | 페이지 구조화 데이터가 매출에 직결 |
| 한국 요실금치료기 카테고리 전담 GEO 도구 글로벌 0개 | GN그룹의 컨설팅 사례가 카테고리 첫 사례가 될 수 있음 |

### 1.2 바디닥터 현재 진단 (요실금치료기 카테고리)

당 연구팀이 자체 측정 도구로 상품 페이지의 AI 가독성을 17개 지표로 자동 산출했습니다 (sixthshop 4영역 배점 체계 차용). _sixthshop 회사가 발표한 공식 점수가 아니라 동일 프레임워크로 직접 계산한 값입니다._

#### 점수가 어떻게 만들어지는지

상품 페이지의 AI 가독성 점수는 **두 가지 출처**가 합쳐져 100점이 됩니다.

| 점수 출처 | 무슨 뜻인가 | 예시 |
|---|---|---|
| **🛒 쇼핑몰이 자동으로 채워주는 부분** | GN코스몰·SSG·11번가 같은 이커머스 플랫폼이 모든 상품에 기본 적용하는 메타정보. 가격·재고·구매 버튼·기본 상품정보 등 | 신상품을 GN코스몰에 등록만 해도 자동 적립 |
| **✍️ GN이 직접 만들어 넣어야 하는 부분** | 상품 설명문·스펙표·인증 표기·FAQ·이미지 alt 텍스트 등 페이지에 사람이 직접 작성해 채워지는 콘텐츠 | GN 마케팅·CS팀이 PDP에 직접 작성 |

이 두 부분이 합쳐져 "총점"이 됩니다. 따라서 **총점이 높다고 해서 AI 최적화를 잘했다는 의미는 아닙니다** — 이커머스 플랫폼 자체의 자동 점수가 큰 비중을 차지할 수 있기 때문입니다. GN그룹이 직접 통제·개선할 수 있는 영역은 ✍️ 부분입니다.

#### 바디닥터 PDP 점수 분해

| 영역 | 만점 | 점수 | 어디서 왔는가 |
|---|---|---|---|
| Schema (상품 정보 구조화) | 35 | **25** | 🛒 25점 — GN코스몰이 자동으로 적립 |
| Content (설명·FAQ·비교 콘텐츠) | 25 | **13** | ✍️ 13점 — GN이 직접 작성 |
| Media (이미지·영상·인포그래픽) | 20 | **6** | ✍️ 6점 — GN이 직접 제작 |
| Commerce (가격·재고·구매·환불) | 20 | **15** | 🛒 15점 — GN코스몰이 자동으로 적립 |
| **총점** | 100 | **59** | 🛒 자동 40점 + ✍️ 직접 19점 |

총점 59점 중 **GN이 직접 만들어 넣은 콘텐츠가 받은 점수는 19점** (45점 만점, 약 42%). 나머지 40점은 GN코스몰 플랫폼이 자동으로 적립한 부분입니다.

#### 컨설팅의 진짜 비교축 — "GN이 직접 만든 콘텐츠 점수"

이 점수만 떼서 보면 **같은 카테고리(요실금치료기) 안에서 누가 페이지 콘텐츠를 잘 만들었는지** 진짜로 비교됩니다.

| 순위 | 브랜드 / 채널 | GN이 직접 만든 콘텐츠 점수 (45점 만점) | % |
|---|---|---|---|
| 1 | **코웨이 테라솔** 자사몰 (메인 PDP) | **33/45** | **73%** ★ 카테고리 1위 |
| 2 | 코웨이 테라솔 자사몰 (상세 페이지) | 31/45 | 69% |
| 3 | 페로니언 11번가 | 30/45 | 67% |
| 4 | 휴온센 11번가 | 28/45 | 62% |
| 5 | 케겔매직 11번가 | 28/45 | 62% |
| 6 | 애플힙 카카오스토어 | 25/45 | 56% |
| 7 | 웨이브케어 자사몰 | 25/45 | 56% |
| 8 | 닥터케이 11번가 | 24/45 | 53% |

| 하위 | 퓨런 자사몰 | 12/45 | 27% |

#### 진단 요약

표면 총점만 보면 "경쟁사도 80점이니 우리도 80점 만들면 차별이 없는 것 아닌가"라고 받아들이실 수 있습니다. 그러나 그 80점의 절반 이상은 **이커머스 플랫폼이 자동으로 적립해주는 점수**(예: 콜게이트 11번가 77점 = 11번가 자동 45점 + 직접 콘텐츠 32점)이며, 해당 브랜드가 페이지를 잘 만들어서 받은 점수가 아닙니다.

진짜 차별을 만드는 것은 **GN이 직접 만든 콘텐츠 점수**이고, 이 영역에서 바디닥터는 카테고리 중하위(42%)이며 코웨이 테라솔(73%)이 카테고리 최고 수준입니다. 본 문서 §2의 P0 6개 항목 적용 시 콘텐츠 점수가 70%대(코웨이 수준)로 상승하고, 동시에 GN코스몰 자동 점수도 약 +10점(aggregateRating·허가번호 추가 적립) 더해져 **총점 80점대 진입은 자연스러운 결과**가 됩니다.

## 2. 개선 실행 항목 — 당 연구팀이 진행할 작업

각 항목 형식: **무엇을 / 왜 / 누가 / 필요한 GN그룹 협조 / 예상 효과 / 시점**

### P0 — 페이지 직접 개선 (1~4주차)

#### 2.1 JSON-LD Product 풀 스키마 적용
- **무엇**: GN코스몰 바디닥터 PDP에 `<script type="application/ld+json">` 삽입. `@type=Product` + `aggregateRating` + `offers` + `identifier` + `brand`/`manufacturer` 풀세트
- **왜**: ChatGPT Shopping card 트리거 1순위 요건. Azoma·Profound 공통 우선순위 1. 현재 Product 최소형만 적용되어 있어 카드 렌더링 불가
- **누가**: 당 연구팀이 스키마·코드 작성 ([초안 완성됨](diagnosis/bodydoctor_jsonld.json)) → GN그룹 개발팀 또는 협력사가 PDP 템플릿에 삽입
- **GN그룹 협조 사항**:
  1. GN코스몰 PDP 템플릿 수정 권한 (직접 또는 협력사 경유)
  2. 정확한 식약처 허가번호 (예: 제허 XX-XXX 호) — 스키마 `identifier` 필드
  3. 정확한 제조사 법인명 — `manufacturer.name` 필드 (식약처 DB에 GN/제너럴네트 미등록으로, 화이트라벨 여부 확인 필요)
- **예상 효과**: AI 멘션률 +30~60% (Ecomtent 고객 평균)
- **시점**: 정보 수령 후 1주차

> 참고: `@type`은 Schema.org 표준에서 `MedicalDevice`도 가능하지만, 본 프로젝트는 일반 카테고리명 표기 원칙에 따라 **`Product`로 통일**합니다. SEO 측면 손실은 미미하며, 다른 필드(aggregateRating·offers·identifier)가 트리거 효과의 90% 이상을 담당합니다.

#### 2.2 인증 정보 상단 배너 + 본문 명시
- **무엇**: PDP Hero 직하에 "식약처 3등급 + 허가번호 제허 XX-XXX 호 + FDA Class 1·2 등록" 풀 텍스트 노출 + 본문 인증 섹션
- **왜**: AI는 페이지 상단 콘텐츠를 우선 파싱. 인증 정보가 하단·이미지 안에만 있으면 추천 신호로 작동하지 않음. 본 연구의 H5/H6(인증 명시가 추천 확률에 영향) 핵심 가설
- **누가**: 당 연구팀이 카피 작성 ([초안 §2 완성됨](diagnosis/bodydoctor_pdp_copy_초안.md)) → GN그룹 PDP 적용
- **GN그룹 협조 사항**: FDA 등록번호 (Class 1과 Class 2 각각)
- **예상 효과**: 인증 풀 명시 +15~25% (업계 일반)
- **시점**: 1~2주차

#### 2.3 핵심 스펙 표 구조화
- **무엇**: "99단계 강도, 5단계 자동 프로그램, 주파수 XX Hz, 사용 시간 15분/회" 등을 `<table>` 태그로 구조화 (불릿·문단 대비 AI 파싱 정확도가 높음)
- **왜**: 본 연구 H1(HTML 포맷이 파싱 정확도에 영향) 가설의 직접 적용. AI는 표 구조에서 스펙을 정확히 추출
- **누가**: 당 연구팀 카피 → GN그룹 PDP 적용
- **GN그룹 협조 사항**: 정확한 스펙 수치 (현재 마케팅 카피에 모호하게 표기되어 있음)
- **예상 효과**: 스펙형 쿼리(가격·사용 시나리오) 추천률 +20%
- **시점**: 2주차

#### 2.4 사용 시나리오·대상 본문 삽입 (Rufus SPN 5 facet)
- **무엇**: PDP 본문에 다음 5개 카테고리 자연스럽게 녹여 작성:
  - **속성**: "간편한, 안정적인, 무소음"
  - **상황**: "출산 후 회복기, 갱년기 케어"
  - **활동**: "앉아 있는 동안 자동 운동, TV 시청 중 사용"
  - **목적**: "골반저근 강화, 요실금 예방·관리"
  - **대상**: "출산 후 여성, 40~60대 여성"
- **왜**: Amazon Rufus 논문 기준 효과 크기 약 2배. 증상·사용 시나리오 쿼리 매칭률 직접 향상
- **누가**: 당 연구팀 카피 ([초안 §4 완성됨](diagnosis/bodydoctor_pdp_copy_초안.md)) → GN그룹 PDP 적용
- **GN그룹 협조 사항**: 페이지 수정 권한만 필요 (추가 정보 불요)
- **예상 효과**: 증상·사용 시나리오 쿼리 +50%
- **시점**: 2~3주차

#### 2.5 FAQ 섹션 + JSON-LD FAQPage 스키마
- **무엇**: PDP 하단에 5~10개 Q&A — "임산부 사용 가능 여부", "식약처 3등급의 의미", "이지케이와의 차이" 등 + FAQPage 스키마
- **왜**: AthenaHQ·Scrunch 공통 권장. AI가 FAQ를 직접 인용해 답변을 생성
- **누가**: 당 연구팀 질문 초안 → GN그룹 직원 인터뷰 후 답변 작성 → 당 연구팀 검수 → 적용
- **GN그룹 협조 사항**: CS팀 인터뷰 1시간 (실제 자주 받는 질문 청취)
- **예상 효과**: FAQ형 쿼리 +25%
- **시점**: 3주차

#### 2.6 비교 콘텐츠 ("바디닥터 vs 이지케이 비교표")
- **무엇**: PDP 안에 등급·가격·차별점 비교 섹션 직접 삽입
- **왜**: Recomaze·Goodie 공통 권장. 비교형 쿼리에 AI가 즉시 인용
- **누가**: 당 연구팀 작성 ([초안 §5 완성됨](diagnosis/bodydoctor_pdp_copy_초안.md)) → GN그룹 영업팀 검수 (경쟁사 표기 톤 점검)
- **GN그룹 협조 사항**: 경쟁사 비교 콘텐츠 게재 가능 여부 확인 (객관 사실 기반만 게재)
- **예상 효과**: 비교형 쿼리 +30%
- **시점**: 3~4주차

### P1 — 외부 콘텐츠 + 이미지 (5~12주차)

#### 2.7 리뷰 캠페인 — 50개 이상 / 평점 4.0 이상
- **무엇**: 기존 구매고객 대상 PDP 리뷰 작성 요청 (쿠폰·샘플 보상)
- **왜**: ChatGPT Shopping rich card 트리거 핵심 요건은 `aggregateRating`. 리뷰 0개로는 카드 렌더링이 거의 불가능. 외부 증거가 페이지 자체보다 강한 추천 신호 (본 연구 H14)
- **누가**: 당 연구팀이 캠페인 카피·운영 매뉴얼 작성 → GN그룹 마케팅 실행
- **GN그룹 협조 사항**:
  1. 기존 구매고객 DB 또는 발송 채널 (이메일·SMS)
  2. 리뷰 보상 예산 약 10만원 (쿠폰 또는 샘플 100건 기준)
  3. 공정거래위원회 표시광고법 준수 가이드 — 당 연구팀이 작성 후 "리뷰어에 쿠폰 보상이 제공되었습니다" 표기 의무화
- **예상 효과**: rich card 렌더링 가능 시 +300% (Alhena 사례)
- **시점**: 5~8주차

#### 2.8 외부 콘텐츠 시딩 (블로그·맘카페·Reddit)
- **무엇**: 협찬·PR 표기를 명시한 후기 콘텐츠를 다음 채널에 분산 게재:
  - 맘스홀릭·시크하우스 등 한국 육아 커뮤니티 — "산후 요실금 후기" 시리즈
  - 네이버 블로그 — 사용 가이드·비교 후기
  - Reddit r/Korea — 영문 후기 (해외 AI 학습 편향 대응)
- **왜**: 본 연구 H14(외부 증거 → 추천 ↑) 가설. AI는 자사몰 외 제3자 채널 언급을 추가 신뢰 신호로 사용. 페이지 단독 개선만으로는 한계가 있음
- **누가**: 당 연구팀이 콘텐츠 가이드·운영 → GN그룹 예산 + 일부 작성자 섭외
- **GN그룹 협조 사항**:
  - 외부 시딩 콘텐츠 예산 (협찬 표기 콘텐츠 5~10건, 별도 협의)
  - 협찬 표기 가이드 — 당 연구팀 작성 후 GN그룹 검수
- **예상 효과**: Evertune 사례 약 +20% (한국 특화 미검증, 보수적 +10~20%)
- **시점**: 6~12주차

#### 2.9 고품질 제품 이미지 재촬영
- **무엇**: 메인·라이프스타일·세부 부품·사용 장면 각 10장 이상 + alt 텍스트 100% 작성
- **왜**: 현재 alt 비율 35% 수준 — AI는 이미지 alt를 콘텐츠로 인식. Media 점수 6/20에서 18/20 가능
- **누가**: GN그룹 촬영 → 당 연구팀 alt 텍스트 작성·검수
- **GN그룹 협조 사항**: 촬영 예산 (사진작가·모델 등, 별도 견적)
- **예상 효과**: 이미지 검색·시각형 쿼리 노출 향상
- **시점**: 4~8주차

#### 2.10 AI 크롤러 방문 로그 분석 인프라
- **무엇**: GPTBot·OAI-SearchBot·PerplexityBot 등 AI 크롤러의 GN코스몰 방문 횟수 모니터링 → 개선 효과의 **선행 지표**
