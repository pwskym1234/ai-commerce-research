# 같은 식약처 3등급 안에서의 경쟁 — 가설 의미 재정의

## 발견

이지케이(EASY-K) 정보 직접 검증 결과:
- 제조: ㈜알파메딕 / 유통: ㈜코리아테크 / 자사몰: easyk.kr
- **식약처 3등급, 허가번호 제허15-329호, 비이식형 요실금 신경근 전기자극장치**
- 99단계 강도 / 5단계 자동 프로그램 / 김상중 광고 모델

→ 우리 바디닥터도 같은 식약처 3등급 + 같은 품목 카테고리 + 99단계.

## 가설 H5/H6/H7의 의미가 정밀해짐

원래 H5: "인증 정보가 명시적일수록 추천 확률 높음"
이는 묵시적으로 "인증 받은 vs 안 받은"을 비교하는 것처럼 읽힘.

새 의미: **"같은 3등급 의료기기 사이에서, 페이지에서 등급+허가번호를 더 명확히 표기하는 쪽이 추천 확률 높음"**

같은 등급 안에서의 차별축:
1. 등급/허가번호 텍스트 표기 (vs 단순 "의료기기" 표시)
2. 인증 정보의 페이지 위치 (상단 배너 vs 하단 푸터)
3. 임상 데이터 인용 유무 (H7)
4. 셀럽 광고 모델 효과 (이지케이 = 김상중)
5. 가격 (직접적인 H 아니지만 control)

## 본실험 가상 페이지 설계에 미치는 영향

산공통 가상 페이지에서 F4(인증 위치) × F5(인증 상세) 수준을 다음과 같이 더 정밀히 정의:

| 수준 | 원래 의도 | 정밀화된 의도 |
|------|---------|------------|
| F5 = 없음 | 인증 표기 X | "의료기기" 단어조차 없음 |
| F5 = 인증완료 | 단순 표기 | "식약처 인증 의료기기" 텍스트만 |
| F5 = 등급+기관+번호 | 상세 표기 | **"식약처 3등급 의료기기, 허가번호 제허**-***호"** 풀 텍스트 |

이지케이의 실제 사례(허가번호 명시 가능)가 우리 가상 페이지의 "최상" 수준 reference가 됨.

## H7 (임상 인용) 의미

원래: "임상 데이터 인용 > 사용자 후기"
정밀화: "같은 의료기기 카테고리에서 임상 RCT/논문 인용이 셀럽 광고+사용자 후기보다 추천 ↑인가?"

→ 이지케이가 김상중 광고 모델 중심이고, 바디닥터 페이지에도 임상 인용 미발견. **양쪽 다 마케팅/후기 중심**.
→ 본실험 가상 페이지에 임상 데이터 인용 변형을 넣으면, 실제로 둘 다 사용 안 하는 전략을 우리만 도입했을 때의 추천율 변화를 측정 가능.
