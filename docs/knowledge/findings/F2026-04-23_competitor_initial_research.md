# 1차 경쟁사 조사 결과

## 1. 의료기기 버티컬 — 핵심 발견

### 1.1 시장이 두 층으로 갈림 (★ 중요)

다나와 "케겔운동기구" 카테고리 상위 1~10위는 **모두 비의료기기 운동 도구** (4,410원~19,800원, 의료기기 표기 없음). 한편 의료기기 인증 EMS 케겔/요실금치료기는 **다른 시장** (20만~50만원대, 자사몰/홈쇼핑 중심).

**시사점**: 일반 사용자가 "케겔운동기 추천"을 AI에게 묻는 쿼리에서 두 시장이 섞여 응답될 가능성 높음. → AiEO 가설 후보:
- **EXPLORATORY_NEW1**: AI는 "케겔운동기" 일반 쿼리에서 의료기기와 비의료기기를 구분해서 추천하는가? 구분 못 한다면 가격이 1/30인 비의료기기가 "추천"에 섞여 들어가서 의료기기 브랜드의 추천 점유율이 희석됨.
- **EXPLORATORY_NEW2**: 쿼리에 "의료기기" 키워드를 명시하면 추천이 정확히 의료기기로만 좁혀지는가?

이 발견은 마스터 §4.4의 H1~H9에 없는 것. 산공통 사전 가설로 추가 검토 필요 (Wayne 결정).

### 1.2 이지케이(EASY-K) 확정

- 공식: easyk.kr (현재 fetch ECONNREFUSED — Wayne 검증 요)
- 식약처 GMP 적합 인증, 5단계 + 99단계 강도, 자동 케겔
- 롯데홈쇼핑 런칭 보도 (k-health.com 2017)
- 바디닥터의 가장 정면 경쟁자. 본실험 필수 포함.

### 1.3 "닥터케이"는 명확한 단일 브랜드로 잡히지 않음

- 가능성: (a) "닥터케겔" 등의 변형 (b) 여러 OEM의 통칭 (c) Wayne 메모 오기
- → Wayne 확인 필요. URL 1~2개 공유 시 확정

### 1.4 OEM 풀의 존재

식약처 emed.mfds.go.kr에 "골반저근자극기" / "요실금치료기" / "전기자극치료기"로 등록된 의료기기를 직접 검색하면 다수 제조사가 나올 것으로 추정. **Wayne이 이 검색을 한 번 돌리면 본실험 비교군 확정에 결정적**.

## 2. 가글 버티컬 — 핵심 발견

확정된 4대 직접 경쟁자:
- **리스테린** (J&J, 글로벌 권위)
- **가그린** (동아제약, 1982 국내 최초, dmall.co.kr)
- **2080** (애경산업, 가성비)
- **페리오** (LG생활건강, 토탈 오랄케어)

프로폴린스 자사 페이지(propolinse.co.kr) 1차 진단:
- F1 추정: 이미지 70% / 불릿 25% / 문단 5%
- F2 JSON-LD: 미발견
- F3 수치 구체성: 높음 ("17개국 4,000만병", "70매", "600ml")
- F6: 시각 시연 중심 ("찌꺼기가 눈에 보임") — 텍스트 LLM에는 약하게 전달될 위험

→ **가설 후보**: 프로폴린스의 USP("시각화")가 AI 추천에서 평가절하될 가능성. JSON-LD + 단백질 흡착 메커니즘 텍스트 명시 시 추천율 상승 예상.

## 3. ✅ 마스터 문서 사실 확인 — 2026-04-23 해결

### 3.1 바디닥터 인증 등급 — 확정

마스터 [PROJECT_MASTER.md §2.2](../../PROJECT_MASTER.md) 원본:
> 바디닥터 요실금치료기 — 식약처+FDA **2등급** 인증

**확정 결과** (서울신문 2024-12-26 + GN코스몰 카테고리 + 검색 종합):
- 식약처: **3등급 조합 의료기기** ("비이식형 요실금 신경근 전기 자극 장치" + "저주파 의료용 조합 자극기")
- FDA: **1등급 + 2등급 모두 등록**
- 99단계 저주파 자극

→ 마스터 §2.2 정정 완료 (2026-04-23). 정확한 식약처 허가번호는 공공데이터 API로 별도 조회 (다음 task).

### 3.2 부수적 발견 — 페이지 자체에 인증 미표기 (★ 컨설팅 핵심)

GN코스몰 바디닥터 카테고리 페이지에서:
- 12개 제품 중 **EMS 허리벨트만 "미국 FDA등록" 명시**
- **요실금치료기 본 제품에는 카테고리/메인 페이지에 인증 표기 없음**
- 임상 데이터 인용도 서울신문 기사 등에 없음

→ **3등급 의료기기인데 그 사실을 페이지에서 강하게 안 알리고 있음**. AiEO 관점 잠재 손실 영역. 컨설팅 Before/After의 핵심 후보. 자세히는 [competitors/bodydoctor.md](../competitors/bodydoctor.md) 참조.
