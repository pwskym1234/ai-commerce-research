# 식약처 의료기기 OEM 풀 13개 확보

## 1. API endpoint 확정

- **Endpoint**: `https://apis.data.go.kr/1471000/MdlpPrdlstPrmisnInfoService05/getMdlpPrdlstPrmisnList04`
- **파라미터**: `serviceKey`, `pageNo`, `numOfRows`, `type=json`, `prduct`(품목명 소문자), `entrps`(업체명 소문자)
- **응답 예시**: `{"body":{"totalCount":209524,"items":[{"item":{"ENTRPS":...,"PRDUCT":...,"PRDUCT_PRMISN_NO":...,"PRMISN_STTEMNT":...,"PRMISN_DT":...}}]}}`
- 코드: [crawler/scripts/mfds_medical_device_api.py](../../../crawler/scripts/mfds_medical_device_api.py)
- 저장 데이터: `data/external/mfds_medical_device/*.jsonl`

**endpoint 발굴 과정**:
- Wayne 제공 baseUrl이 단서였지만 operation 이름 `getMdlpPrdlstPrmisnList04`를 data.go.kr 페이지 HTML raw grep으로 추출
- 파라미터는 대문자(`PRDUCT`)가 아니라 소문자(`prduct`)라는 점이 핵심
- 공식 명세 PDF 없이 확정. Swagger UI가 JS 렌더링이라 WebFetch 한계 있었음

## 3. ★ 바디닥터는 DB에 "제너럴네트" 법인으로 없음

다음 변형 모두 `entrps` 필터 0건:
- 제너럴네트 / (주)제너럴네트 / 지앤메디 / (주)지앤메디 / GN바디닥터 / 네트 / 지앤 / 바디닥터

품목명 "바디닥터" 포함도 0건.
마스터 §2.2에 언급된 "저주파 의료용 조합 자극기" 카테고리로도 0건 (다른 이름일 수도).

**해석 (유력)**: 바디닥터 = **화이트라벨/OEM 제품**. 제조·허가는 다른 등록업체가 하고 GN그룹(제너럴네트)은 수입/유통/마케팅만.

후보 시나리오:
- (a) 알파메딕 등이 제조, GN이 재브랜딩 — 그러면 두 브랜드 스펙 같음 설명됨 (99단계 공통)
- (b) 다른 제조사 (썬텍·청우·퓨런 등)에서 제조
- (c) 해외 OEM + 국내 수입허가 (수허)

**검증 방법**: GN코스몰 제품 상세 페이지 최하단의 **"상품정보 표시(일반상거래법)"** 섹션 확인. 여기에 제조사/원산지/허가번호가 법적으로 명시되어야 함.

→ Wayne 액션: 해당 섹션 캡처해서 공유 또는 제조사 명 알려주기.

## 4. ★★ 본실험 경쟁군 N=6 권고 업데이트

13개 OEM 풀에서 다음 기준으로 본실험 경쟁 2~3종 선정:

| 기준 | 후보 |
|------|------|
| 시장 레거시 (오래 등록) | **썬텍메디칼** (수허 12-749, 수입) — 수입 채널 대표 |
| 최근 진입 (최신) | **(주)세라젬** (23-785) — 대기업. 판매 채널 강함 |
| 중견 + 다수 모델 보유 | **퓨런헬스케어** (2건) 또는 **청우메디칼** (2건) |
| 직접 경쟁 확정 | **알파메딕 EASY-K** (제허 15-329) |

**본실험 의료기기 N=6 (최종 권고)**:
1. **바디닥터** (우리, anchor)
2. **알파메딕 EASY-K** (제허 15-329)
3. **세라젬** (제허 23-785) — 대기업 브랜드 인지도
4. **퓨런헬스케어** (제허 21-492 또는 21-920) — 중견
5. **비의료기기 노이즈 1종** (애플힙 등)
6. **해외 1종** (Elvie)

→ N=4의 "인증 받은" 표본 확보 (우리 + 이지케이 + 세라젬 + 퓨런) = H5 검정에 충분

## 5. 가설에 미치는 의미

### H10 (AI가 의료기기/비의료기기 구분?) 강화
13개 등록 업체 vs 다나와 카테고리 1~10위의 만원대 비의료기기 운동도구 — 명확한 경계. AI가 이를 텍스트에서 구분하는지 본실험 측정 가능.

### H5/H6 새 해석
같은 3등급 카테고리 13개 내에서 **AI 추천 받는 브랜드 vs 못 받는 브랜드 차이**가 AiEO의 진짜 타겟:
- 세라젬(대기업) — 인지도
- 알파메딕(전문) — 임상 근거
- 우리 바디닥터 — 페이지 최적화

본실험에서 이 3가지 브랜드 전략이 AI 추천에 어떻게 작용하는지 측정.
