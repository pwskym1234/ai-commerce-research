# 의료기기 EMS 케겔 OEM 풀 (개별 미확정)

> 단일 브랜드 파일 아님. "의료기기 인증 받은 EMS 케겔/요실금 자극기" 카테고리 내 OEM 다수 존재 가능성을 표시하는 풀 파일.

## 배경
- 식약처 emed.mfds.go.kr에서 "골반저근자극기" / "요실금치료기" / "전기자극치료기"로 등록된 의료기기를 검색하면 다수의 제조사가 존재할 것으로 추정 (검증 필요)
- 네이버 스마트스토어에서 "요실금치료기 의료기기 2등급"으로 검색 시 중소 OEM 브랜드가 혼재 가능
- 대부분 중국 OEM 제조 + 국내 식약처 인증만 부착하는 형태일 가능성 (추정)

## 다음 액션 (Wayne)
1. **emed.mfds.go.kr 직접 검색** — "요실금" 또는 "골반저근" 품목 → 제조사·모델 리스트 (공공데이터)
2. 상위 5~10개 추출 → 그중 네이버/쿠팡 판매 활성 1~2개를 추가 stub 작성
3. 본실험 비교군에 "의료기기 인증 但 마이너 브랜드" 1~2개 포함 → AI가 인지도 vs 인증 중 무엇을 가중하는지 측정 (H5/H8 관련)

## "닥터케이" 참고
- Wayne 언급 "닥터케이"는 직접 검색에서 명확한 단일 브랜드로 잡히지 않음
- 가능성: (a) "닥터케겔" 등의 변형 (b) 여러 OEM의 통칭 (c) Wayne 메모 오기
- → Wayne 확인 필요. URL 1~2개 공유 시 확정 가능

## 카테고리 분리
주의: 다나와 "케겔운동기구" 카테고리(만원대)와 "요실금치료기/EMS 의료기기"(20~50만원대)는 **다른 시장**. AI에게 쿼리할 때 둘이 섞여 응답될 가능성이 본 실험의 핵심 관찰 포인트.
