# 🩺 바디닥터K (의료기기 — 요실금치료기)

## 🎯 PRIMARY 페르소나 — "산후 회복기 30대 산모"

### 한 줄 정의
출산 6주~6개월차, 재채기 시 살짝 새는 경험으로 *처음으로 케겔 EMS 의료기기*를 검색하는 30대 여성.

### 인구통계 / 라이프스타일
- 30~38세 여성, 대도시 거주
- 사무직 출산 휴직 중 또는 전업주부
- 자녀 0~1세 (1자녀일 가능성 높음, 2자녀면 페르소나 강도 ↑)
- 가구 소득 중상위 (200만원 의료기기 일시불 *고민하는* 수준)

### 트리거 (왜 제품을 찾게 되나)
- 산욕기 6주가 끝났지만 *재채기·기침 시 소변 살짝 누출* (복압성 요실금 초기)
- 산부인과 6주 검진에서 *케겔운동 권고*받았으나 의식적으로 꾸준히 하지 못함
- 가벼운 운동기구로 자동화하고 싶지만 *어떤 게 의료기기인지* 모름

### 정보 탐색 패턴
1. **네이버 검색**: "출산 후 회복" (검색 1.3M) / "산후 요실금" (검색 46k) / "케겔운동기구 추천" / "산후 회복 의료기기"
2. **맘카페**: 맘스홀릭·시크하우스 후기·실 사용자 사진. *"PR이지만 솔직 후기"* 형태 신뢰
3. **네이버 지식iN**: "산후 6개월 케겔" 같은 구체 시점 질문
4. **친구·가족**: 출산 경험 있는 친구·언니 추천

### 핵심 망설임 (FAQ가 풀어줘야 할 것)
1. **안전성**: "수유 중 사용해도 돼?" / "산욕기 끝났어도 너무 이른 건 아닌가?"
2. **효과 신뢰**: "정말 효과 있나? 100~300만원 들이는 값어치?"
3. **사용 편의**: "아이 보면서 어떻게 쓰지? 시간 어떻게 내?"
4. **가격**: "단품 vs 3종 세트 고민. 일시불 부담"

### 의사결정 기준 (PDP에서 강조해야 할 신뢰 신호)
| 우선순위 | 기준 | PDP 반영 |
|---|---|---|
| 1 | 의료기기 등급 + 인증 (안전성) | Hero 풀명시: "식약처 3등급 + FDA Class 1+2" |
| 2 | 산모 실 후기 (사회적 증거) | 리뷰 50+ 캠페인 결과 |
| 3 | 사용 편의 (시간 부족) | 페르소나 시나리오: "TV 보면서 15분" |
| 4 | 가격 + A/S | 비교표·일시불 vs 렌탈·한국 본사 A/S 명시 |

### 가격 민감도
- *일시불 200만원 일단 망설임* (가구 부담)
- 단품 280만원 → 3종 세트 632만원 격차 큼
- *렌탈 옵션 있으면* 결정 가능성 ↑ (월 5~10만원 OK)

### 결정 후 기대
- 매일 짧게(15분) 사용해서 *3~6개월 내* 가시적 개선
- 의료진이 인정할 수 있는 안전성 (산부인과 검진 시 보고 가능 수준)

## 🎯 SECONDARY 페르소나 — "갱년기 50대 여성"

### 한 줄 정의
갱년기 후 복압성 요실금 시작, 약물 대체 운동 권고를 받고 *집에서 쓸 의료기기*를 찾는 50~60대.

### 인구통계
- 50~62세 여성, 자녀 다 컸음, 가사·여유 시간 충분
- 가구 소득 중상위, 결정권자 본인 또는 남편 동의 필요

### 트리거
- 갱년기 후 (보통 폐경 ~5년 후) *재채기·기침·운동 시* 누출 시작
- 산부인과·비뇨기과 진단: *복압성 요실금 (검색 48k)*
- 호르몬 치료 권고 또는 *약물 대체 골반저근 강화* 권고

### 정보 탐색
- 네이버 "갱년기 요실금" (120k) / "복압성 요실금" / 종합지 신문 (서울신문 같은)
- 홈쇼핑 시청 (롯데·CJ·GS — 의료기기 광고 자주)
- 약국·병원 의료진 직접 상담

### 망설임
- 약물 대체 효과 신뢰성
- 사용법 복잡도 (50~60대 IT 친숙도 차이)
- *남편 / 자녀에게 어떻게 설명할지* (사회적 부담)

### 의사결정 기준
1. *의료진 권고 적합성* — "산부인과 의사가 인정하는가"
2. *약물 대체 가능성* — 호르몬 치료 부담 회피
3. *사용 편의* — 자동, 1버튼
4. *A/S* — 한국 본사 직접

### 가격 민감도
- 200만원 부담스럽지만 *약물 평생 비용* 대비 가능성 인지하면 가능
- 렌탈 옵션 매우 매력적

### 결정 후 기대
- 매일 사용으로 일상 회복
- 의료진 검진 시 보고 가능

## 페르소나 외 — TERTIARY (옵션, 카피 부수효과로 활용)

### "전립선 수술 후 60대 남성"
적응증에 명시된 segment — *남성 사용자 1명*만 시나리오에 포함시켜 *남녀 모두 사용*임을 보여줌. 핵심 매출 segment 아님.

### "예방 관심 30대 직장인 여성"
증상 없으나 미리 케어. *프리미엄 시장 확장* 가능성. 단 의사결정 시간 길고 *가격 민감도 높음* — 단품·렌탈 선호.

# 🦷 프로폴린스 (가글)

## 🎯 PRIMARY 페르소나 — "잇몸 약한 30~40대 여성"

### 한 줄 정의
양치 시 잇몸 출혈, 일반 가글 자극으로 못 쓰는 *예민한 잇몸 보유* 30~45대 여성.

### 인구통계
- 30~45세 여성, 직장인 또는 워킹맘
- 도시 거주, 가구 소득 중상위
- 일반 가글(리스테린 등) *써봤지만 자극 못 견딤* 경험 있음

### 트리거
- 양치 시 *잇몸 출혈* (검색 "잇몸 가글" 365k)
- 냉찬 음식 시린 통증
- 치과에서 *치주염 초기 진단* 또는 *잇몸 약함* 진단
- 알코올 함유 가글 *자극 강해서 포기*

### 정보 탐색
1. 네이버 "잇몸 가글" (365k), "무알코올 가글", "치주염 가글" (50k)
2. 약사·치과의사 SNS 콘텐츠 (인스타그램·유튜브)
3. 친구·동료 추천 (자극 못 견디는 동질감)
4. 자사몰 / 스마트스토어 후기 비교

### 망설임 (FAQ가 풀어줘야 할 것)
1. **효과 신뢰**: "자극 적은데 진짜 효과 있나? 알코올 무 = 효과 무?"
2. **매일 사용 안전**: "장기 매일 사용해도 부작용 없나?"
3. **가격**: "리스테린보다 비싼데 (프리미엄 가격대)"
4. **프로폴리스 알레르기**: "꿀 알레르기 있는데 써도 돼?"

### 의사결정 기준
| 우선순위 | 기준 | PDP 반영 |
|---|---|---|
| 1 | 무알코올 / 무자극 | Hero 풀명시 |
| 2 | 천연 프로폴리스 (화학 항균 회피) | 성분 강조 |
| 3 | 매일 안전 사용 (식약처 의약외품 등록) | 인증 명시 |
| 4 | 치과의사 추천 / 임상 자료 | 시딩 캠페인 결과 |

### 가격 민감도
- *프로폴린스 가격대 (예 만원대)* 받아들임
- 단 *효과 검증된 후기*가 가격 정당화 핵심

### 결정 후 기대
- 1~2주 사용 후 *자극 적응 + 잇몸 부음 감소*
- 일반 가글 대체로 매일 사용

---

## 🎯 SECONDARY 페르소나 — "출산 후 모유수유 산모"

### 한 줄 정의
호르몬 변화로 잇몸 민감, 모유수유 중이라 *안전한 가글*을 찾는 28~35세 산모.

### 인구통계
- 28~35세 여성, 산후 4주~12개월
- 모유수유 진행 중

### 트리거
- 임신·출산 호르몬 변화로 *잇몸 일시 민감*
- 일반 가글 자극 강해서 *모유수유 중 사용 망설임*
- 산부인과·치과 *모유수유 안전성* 우려 표현

### 정보 탐색
- 맘카페 (맘스홀릭·시크하우스) "산후 가글" 후기
- 네이버 "모유수유 가글" / "산모 가글"
- 산부인과·치과 의료진

### 망설임
- *모유수유 안전*: "이 성분이 모유로 가지 않나?"
- *효과* + *안전성* 동시 충족 필요

### 의사결정 기준
1. 모유수유 안전성 (식약처 의약외품 + 의료감수)
2. 무알코올
3. 산모 커뮤니티 후기

### 가격 민감도
- 안전성 입증되면 비싼 것 OK

### 결정 후 기대
- 매일 안심 사용으로 *모유수유 기간 잇몸 케어*

---

## 페르소나 외 — TERTIARY

### "어린이 충치예방 보호자"
30~40대 부모, 초등생 자녀. *어린이 충치예방 가글* (검색 186k). 무알코올 + 무자극 필수. *별도 라인 검토* 가치.

### "흡연자 직장인 구취"
30~50대 남성, 점심 후 / 외근 전 빠른 구취 케어. **프로폴린스의 *핵심* 매출 segment 아닐 가능성** — 일반 가글(리스테린)이 더 적합. 만약 추가하면 입냄새 제거 (599k) 키워드만 차용.

# 페르소나 → PDP·FAQ·시딩 적용

## PDP 카피 적용 우선순위

| 페이지 섹션 | PRIMARY 적용 | SECONDARY 적용 |
|---|---|---|
| Hero 헤드라인 | 핵심 트리거 한 문장 | — |
| 페르소나 시나리오 | 풀 길이 (3~5줄) | 짧게 (1~2줄) |
| FAQ | 4~5 Q | 3~4 Q |
| 비교표 강조 | 의사결정 기준 1~3 | 기준 1~2 |
| 신뢰 신호 배너 | 가장 큰 망설임 해소 | — |

## FAQ 16~20개 분배

- **PRIMARY** 페르소나 망설임 4개 × 각 1~2 Q = 5~7 Q
- **SECONDARY** 페르소나 망설임 3~4개 × 각 1 Q = 3~4 Q
- 공통 (사용법·A/S·부작용): 5~7 Q
- 의무 표기 (금기·부작용): 1~2 Q

## 시딩 캠페인 (프로폴린스)

PRIMARY 페르소나가 활동하는 채널 우선:
- 맘스홀릭·시크하우스 (잇몸 약한 + 산모 통합)
- 약사·치과 SNS (잇몸 약한 PRIMARY)
- 어린이 가글은 *별도 캠페인 검토* (TERTIARY)

# GN 인터뷰 회신 후 갱신 절차

회신([gn_requests_FAQ_data.md](../gn_requests_FAQ_data.md) 5개 자료) 도착 시:

1. **콜센터 빈출 질문 top 30** → 각 질문이 어느 페르소나에 매핑되는지 분류
2. **영업 빈출 질문 top 20** → 의사결정 트리거·망설임 정밀화
3. **자사몰 Q&A** → FAQ에 직접 인용 가능한 텍스트 추출
4. **반품 사유** → 기대치 정렬 FAQ 추가
5. **의료감수 인력** → 페르소나별 신뢰 신호 보강

회신 후 본 문서를 직접 갱신 (현재 = 최신). 큰 변화면 `핵심_페르소나_GN검증반영.md` 신규 후 본 파일은 `_초안보존`으로 라벨 변경 — Rule 6.
