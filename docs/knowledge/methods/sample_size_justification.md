# 표본 크기 정당화 — N=6, 반복 20회

## 0. 결정

본실험 설계 (2026-04-23 확정):
- **비교군 N = 6** (의료기기 6, 가글 6)
- **반복 = 20회** (조합당 × 쿼리당)
- **조합 = 36** (L36 직교배열)
- **쿼리 = 4** (Q1 4유형)
- **LLM = 1 (ChatGPT 단일)** ← 2026-04-23 Wayne 결정. 비용 1/3로 축소.
- **총 호출 = 36 × 4 × 20 × 1 = 2,880회** (~$29 gpt-4o)
- 발표 전 시간 남으면 핵심 가설(H1, H8)만 Gemini/Perplexity sanity check (별도 +몇백 호출)

## 1. 학술 표준에서의 N (브랜드 수) 비교

| 분야 | 통상 N | 우리 매칭 |
|------|-------|---------|
| AI 시스템 비교 (HELM, MT-Bench 등) | 5~30+ | N=6 ✅ |
| 마케팅 conjoint analysis (Green & Srinivasan 1990) | 4~8 stimuli/set | N=6 ✅ |
| 의료기기 사용자 선호 비교 | 3~6 brands | N=6 ✅ |
| 소비자 선택 모델링 (McFadden) | 3~10 alternatives | N=6 ✅ |

→ N=6은 **표준적 수준**. "표본 적다" critique 정면 대응 가능.

## 2. 진짜 데이터 양 — N이 아니라 (조합 × 쿼리 × 반복)

본실험 row 수:
- **산공통 (Y2 binary 추천 여부)**: 36 × 4 × 20 × 1 = **2,880 row**
- **데마 (실제 크롤링 + 페이지 피처 + 리뷰)**: 6 브랜드 × 4 채널 × 평균 100 리뷰 ≈ **2,400+ row** (페이지 피처 별도 추가)

학술 표준 비교:
- **회귀분석**: variables당 10~20 cases (Harrell's rule). 6요인 + 교호작용 → 100~200 row 최소. **2,880은 충분** (LLM=1 기준).
- **머신러닝 학습**: 통상 1,000+ row 권장. **2,400+ 충분**.
- **이항 비율 검정**: 셀당 n=15~20이면 95% CI 폭 ≈ ±0.20. **n=20으로 OK**.

## 3. N=6의 의미 — "데이터 양"이 아니라 "Y2 base rate"

N개 중 무작위 선택 시 베이스 추천 확률 = 1/N.

| N | 베이스 | 효과 검출 안정성 |
|---|------|---------------|
| 2 | 50% | 변동성 큼, 효과가 약하면 묻힘 |
| 4 | 25% | 안정 |
| **6** | **17%** | **안정 + base rate에서 ±10pp 효과 검출 깔끔** |
| 8 | 13% | 매우 안정, 시각화 약간 복잡 |
| 10+ | ≤10% | 시각화/분석 복잡 |

N=6은 base rate ↔ 효과 크기 검출의 sweet spot.

## 4. 반복 횟수 = 20의 정당화

이항 비율의 95% Wilson 신뢰구간 폭 (n=20, p=0.5 기준):

| 반복 n | 95% CI 폭 |
|--------|---------|
| 10 | ±0.30 |
| 15 | ±0.24 |
| **20** | **±0.21** |
| 30 | ±0.18 |
| 50 | ±0.14 |
| 100 | ±0.10 |

**판정**: n=20이면 ±0.21 — 효과 크기 0.20+이면 검출 가능. 우리가 가설 H1~H11에서 기대하는 효과 크기는 보통 0.10~0.30 범위 → 일부 약한 효과(±0.10)는 n=30+ 필요.

**파일럿 후 결정 옵션**:
- 파일럿(R9)에서 σ 측정 → 본실험 n 결정
- 비용 여유 있으면 n=30 (호출 +50%, ~$130)으로 강화
- 약한 효과 가설에 대해서만 n=30, 강한 효과는 n=20 — 이종 분배

## 5. 일반 데이터 분석 표준과의 비교

| 표준 | 권고 | 우리 |
|------|------|------|
| Cochran (1977) 표본 표 (오차 ±5%) | n ≥ 384 | 본실험 row 8,640 ✅ |
| Cohen (1988) effect size 검정력 분석 (medium effect, α=.05, power=.80) | n ≥ 64/group | 본실험 셀당 n=20 (LLM 1개) ⚠️ → n=30~50이면 충분 ✅ |
| Harrell (2015) 회귀분석 EPV ≥ 10~20 | events ≥ 60~120 (6요인 기준) | 2,880 row ✅ |
| ML 학습 (1,000+ row) | n ≥ 1,000 | 2,400+ ✅ |

**약점 1개**: Cohen power analysis 기준 셀당 n=64. 우리는 LLM 1개에 n=20이라 부족. **약한 효과 검정 가설은 n=30~50으로 조정** (파일럿 후 결정).

## 6. 진짜 신뢰도 약점은 어디서 — 우려 우선순위

| 우려 | 심각도 | 우리 대응 | 강화 옵션 |
|------|------|--------|---------|
| LLM 응답 변동성 (n=20 약함) | 중 | Wilson CI 명시 | n=30~50 (비용 +50~150%) |
| 가상 페이지 ≠ 실제 페이지 | 중 | Phase 4 외적 타당성 검증 | OK (마스터 §4.5) |
| **단일 LLM 일반화 약함** | **중** | 본실험 GPT만. 발표 전 Gemini/Perplexity로 핵심 가설 sanity check | 비용 추가 ~$30 |
| **N=6 자체가 적음** | **낮음** | 표준 학술 N (위 표 §1) | 불필요 |
| Position bias | 낮 | 매 반복 셔플 (R4) | OK |

→ **N은 약점 아님**. 우려는 "반복 n=20"과 "외적 타당성"에 집중.

## 7. 발표 대응 스크립트

청중: "N=6은 너무 적지 않나요?"

답:
> "N=6은 마케팅 conjoint analysis의 표준 stimuli/set 범위(4~8)이며, 실제 데이터 row는 N이 아니라 조합·쿼리·반복의 곱입니다. 본실험은 8,640 row를 산출하며, 이는 6요인 회귀분석 기준 EPV(events per variable) 약 1,440으로 Harrell의 권고 10~20을 압도합니다. 신뢰도 우려는 N보다 LLM 응답 변동성에 있으며, 이를 위해 매 셀 20회 반복 + Wilson 95% CI를 함께 보고합니다."

청중: "반복 20회가 적지 않나요?"

답:
> "이항 비율 95% CI 기준 ±0.21로, 우리가 가설에서 기대하는 효과 크기 0.10~0.30 범위에서 강한 효과는 검출 가능합니다. 파일럿(20조합 × 5회)에서 측정한 표준편차로 본실험 반복 횟수를 결정하며, 약한 효과 검출이 필요한 가설에 대해서는 30~50회로 확장합니다."
