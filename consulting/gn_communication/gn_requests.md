# GN그룹/제너럴네트 측에 확인·요청할 리스트

**작성일**: 2026-04-24
**용도**: Wayne이 GN 본사와의 커뮤니케이션에서 사용할 질문·요청 정리
**배경**: 공개 크롤링·식약처 API·페이지 진단 결과 확인되지 않은 정보 + AiEO 컨설팅 산출물 생성을 위해 필요한 자료

---

## 🔴 P0 — 프로젝트 진행 블로커 (즉시 확인 필요)

### 1. 바디닥터 요실금치료기의 실제 식약처 등록 정보

우리 공개 자료 조사 결과:
- 서울신문 2024-12-26 기사: "식약처 3등급 의료기기"
- 식약처 공공데이터 API에서 "제너럴네트" 법인명으로 검색 0건
- 품목명 "비이식형요실금신경근전기자극장치" 등록 13개 회사 리스트에도 제너럴네트 없음
- GN코스몰 제품 상세 페이지에 제조사/허가번호 법적 고시 섹션 없음

→ **확인 요청**:
- [ ] 바디닥터 요실금치료기의 **정확한 식약처 허가번호** (예: "제허 XX-XXX 호" 형식)
- [ ] **실제 제조사 법인명** (제너럴네트가 제조인지, 수입/유통이고 OEM이 제조하는지)
- [ ] 제조사가 별도라면 OEM 파트너 회사명 (13개 풀 중 하나인지 확인 — 알파메딕/썬텍메디칼/청우메디칼/비엠씨/퓨런헬스케어/쉬엔비/유진플러스/리모트솔루션/엠비랩/세라젬 중)
- [ ] 식약처에 등록된 정확한 **품목명** ("비이식형요실금신경근전기자극장치" 외 다른 카테고리인지)
- [ ] **FDA 등록 번호** (Class 1 + Class 2 각각)
- [ ] 허가 문서(식약처 + FDA) 사본 — 공개 가능 범위

### 2. 임상/효과 근거 자료

현재 서울신문 기사, GN코스몰, 자사 공식 페이지 모두 **임상 데이터 인용 없음**. 경쟁사 검색에서도 드물음.

→ **확인 요청**:
- [ ] 바디닥터 효과에 대한 **임상시험 데이터** 존재 여부
- [ ] 효과 검증 자료(사내 테스트, 의사 감수, 대학병원 협력 등)
- [ ] **"99단계 저주파 자극"의 의학적 근거** (단계 수가 효과에 미치는 영향 연구 자료)
- [ ] 사용자 설문/추적 데이터 (자체 수집 자료)
- [ ] 해외 유사 기기의 RCT 논문을 인용할 수 있는지 (PFMT + EMS 조합)

### 3. 제품 단독 가격 + 판매 채널 실태

- GN코스몰에서 바디닥터 요실금치료기 단독가 2,800,000원, 현재 품절
- 3종 세트 6,320,000원

→ **확인 요청**:
- [ ] 단독 제품의 **정확한 현재 판매가** (여러 채널별)
- [ ] **판매 중단 사유** (품절이 실제 품절인지, 운영 중단인지)
- [ ] 판매 채널별 실제 판매 볼륨 (월/분기)
- [ ] 네이버 스마트스토어(gncosshop)의 실제 리뷰 수 + 평점
- [ ] 11번가/SK스토아/홈쇼핑 판매 중인지 (이지케이/김상중 광고처럼 홈쇼핑 전략)

## 🟡 P1 — AiEO 컨설팅 리포트 생성에 필요 (1주 내)

### 4. 현재 페이지 운영 권한

→ **확인/협의**:
- [ ] GN코스몰(gncosshop.com) 제품 상세 페이지 **수정 권한** 있는지 (Wayne이 직접 수정 vs GN 개발팀에 요청)
- [ ] **네이버 스마트스토어** (smartstore.naver.com/gncosshop) 수정 권한
- [ ] **자사 공식몰 bodydoctor.co.kr** 수정 권한 + 사이트 운영 상태 (DNS resolution 실패 경험)
- [ ] GN에 관련 개발사 혹은 대행사 정보 (수정 요청 주체)
- [ ] JSON-LD/구조화 데이터 추가 가능 여부

### 5. 리뷰 확보 전략

- AI 추천에서 review_count / average_rating 이 핵심 신호 (ChatGPT Shopping, Rufus 모두)
- 현재 GN코스몰 바디닥터 제품 리뷰 **0개**

→ **협의**:
- [ ] 기존 구매 고객에게 리뷰 요청 캠페인 가능한지
- [ ] 전문가 리뷰(요실금 전문 산부인과 의사) 섭외 가능한지
- [ ] 블로그 체험단/인플루언서 프로그램 이력 (있다면 재활성화)

### 6. 외부 인용 경로

AI 추천에서 제3자 인용 (Reddit, 블로그, 뉴스)이 강한 신호. 바디닥터 관련 외부 콘텐츠 기존 이력 확인.

→ **확인**:
- [ ] 서울신문 2024-12-26 기사 외 **언론 보도 이력** (자사 PR 자료)
- [ ] 의료/헬스 전문지 게재 이력 (코메디, 헬스조선 등)
- [ ] 기존 블로그/카페/커뮤니티 콘텐츠 시딩 이력 (맘카페, Reddit 한국)
- [ ] 유튜브 리뷰 영상 (홈쇼핑 방영본 외)

## 🟢 P2 — 본실험·확장 단계에 필요 (2~4주 내)

### 7. 마케팅·예산 협의

- AiEO 컨설팅 결과 실행에 필요 예산 규모
- Reddit/맘카페 시딩, 의료 전문지 기고, 임상 데이터 확보 등 외부 증거 전략

→ **협의**:
- [ ] AiEO 개선 프로젝트 **예산 가이드라인** (Before/After 측정 기간의 운영비 포함)
- [ ] 기존 온라인 마케팅 예산 **일부 재배분 협의** (배너 광고 → 시딩 콘텐츠)
- [ ] **AMP/ACP 프로토콜 대응**(장기) 투자 의향 — OpenAI·Amazon·Google 쇼핑 에이전트 직접 연동

### 8. 제품 고도화 피드백

- 경쟁사 대비 99단계 EMS는 차별 안 됨 (이지케이도 동일)
- 차별 포인트가 페이지에 드러나지 않음

→ **확인**:
- [ ] 바디닥터만의 **고유 기술·특허** 자료 (경쟁사 EASY-K와 구분되는 점)
- [ ] **개발팀의 차세대 제품** 로드맵 (있다면 현 제품 개선 방향 맞춤)
- [ ] **해외 수출 현황** (1000만불 탑 수상 → 수출국별 브랜드 인지도)

### 9. 경쟁 정보

- 식약처 등록 13개사 중 세라젬(대기업) + 퓨런헬스케어(중견)이 본실험 비교군 권고
- 이들의 실제 시장 점유율/판매 데이터는 공개 자료만으로 약함

→ **확인**:
- [ ] 업계 내부에서 아는 **경쟁사별 실제 시장 점유율** (유통·영업이 알 가능성)
- [ ] 경쟁사 대비 바디닥터의 내부 차별 포인트 (영업팀 경험)
- [ ] **해외 경쟁사**(Elvie, Kegel8, PeriCoach 등) 한국 수입 현황
