# 이지케이 (EASY-K)

## 핵심 사실
- **카테고리**: 가정용 EMS 자동 케겔운동 의료기기 (의자/방석에 앉아서 사용)
- **기술**: 5단계 프로그램 + 99단계 강도 조절. 인체에 적합한 전기자극으로 자동 케겔운동 유도
- **사용**: 하루 15분, 4주 이상 사용 시 골반저근 강화 효과 광고
- **인증**: "식약처 GMP 적합 인증" 표현 확인. 의료기기 등급(2/3) 및 허가번호는 **미확인**
- **유통**: 롯데홈쇼핑 런칭 사례 보도. 자사 공식몰(easyk.kr) + 렌탈 옵션
- **포지셔닝**: 바디닥터와 직접 경쟁. 동일 사용 시나리오(앉아서 자동 EMS).
